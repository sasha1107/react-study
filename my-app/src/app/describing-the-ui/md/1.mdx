# Describing the UI

리액트는 UI 렌더링을 위한 자바스크립트 라이브러리이다.
리액트를 사용하면 재사용 가능하고 중첩 가능한 컴포넌트로 UI를 결합할 수 있다.

<br/>

# Your first component 

### 리액트 컴포넌트란?

> 마크업을 뿌릴 수 있는 자바스크립트 함수

<br/>

# 컴포넌트 import, export

하나의 파일에서 여러 개의 컴포넌트를 선언할 수 있지만, 큰 파일은 탐색하기 어려울 수 있다.
이를 위해 컴포넌트를 다른 파일로 분리할 수 있여 `export` / `import` 해 사용할 수 있다.
